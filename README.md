# Android Java – Calcul Impôt Immobilier

## 📖 Description
Cette application Android permet de calculer un **impôt immobilier** à partir de trois informations :

- La **surface du logement** (en m²)
- Le **nombre de pièces**
- La présence d'une **piscine**

L'utilisateur saisit les informations dans l'application puis appuie sur le bouton **Calculer** pour afficher le montant total de l'impôt.

---

## ⚙️ Fonctionnalités
- **EditText** : saisie de la surface et du nombre de pièces.
- **CheckBox** : option pour indiquer la présence d’une piscine.
- **Button** : déclenche le calcul de l’impôt.
- **TextView** : affiche le résultat du calcul.

---

## 🖥️ Exemple d’exécution
- Surface : `200 m²`  
- Nombre de pièces : `4`  
- Piscine : cochée ✅
